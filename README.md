# 🏨 酒店客补预算监控看板

客补预算监控可视化看板，实时展示MTD执行率、补贴率、流量转化比等核心指标。

## 🌐 在线访问

部署后访问地址：`https://你的用户名.github.io/hotel-budget-dashboard`

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 主看板页面 |
| `data.json` | 看板数据（每日更新） |
| `README.md` | 项目说明 |

## 🔄 数据更新

1. 修改 `data.json` 中的数据
2. 提交并推送：`git add . && git commit -m "update data" && git push`
3. 等待2分钟自动部署

## 📊 指标说明

- **MTD执行率**：月度累计预算执行比例，目标100%±1%
- **执行-时序偏差**：实际执行与计划的时间差
- **流量转化比**：拉流量与促转化的结构比
- **补贴率**：客补补贴占GMV比例
